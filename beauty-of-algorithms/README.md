# 📚 数据结构与算法之美 —— 代码实现

> 基于王争老师的极客时间专栏《数据结构与算法之美》，使用 Java 实现所有核心代码案例、课后习题及推荐练习题。

## 📁 目录结构

| `ch49_astar` | 第49讲：A*寻路 | A*启发式搜索（f=g+h 曼哈顿距离评估函数） |
