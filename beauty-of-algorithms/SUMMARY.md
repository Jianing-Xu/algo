# 《数据结构与算法之美》题型归纳与解法沉淀

这个模块按王争《数据结构与算法之美》专栏的章节组织代码，同时把课程推荐的 LeetCode 高频题收录在 `leetcode` 包中。当前共有 45 个源码文件，覆盖课程主线章节与一组配套高频题，目标是把每种数据结构和算法思想的核心套路沉淀成可复用模板。

## 目录结构

- `ch05_array` ~ `ch49_astar`：按课程章节分包，覆盖数组、链表、栈、队列、排序、二分、跳表、LRU、树、堆、图、字符串匹配、Trie、分治、回溯、DP、拓扑排序、最短路径、A*
- `leetcode/`：课程推荐的 LeetCode 必做题，当前覆盖数组、链表、栈、队列、递归、树、哈希、二分、字符串

## 一、核心题型与共性解法

### 1. 数组与双指针

共性套路：排序预处理 + 双指针夹击 / 快慢指针，把 O(n²) 降到 O(n)。

| 题目 | 核心技巧 | 面试频率 |
|------|----------|----------|
| LeetCode 15 三数之和 | 排序 + 固定一端 + 左右夹击去重 | ⭐⭐⭐⭐⭐ |
| LeetCode 41 缺失第一个正数 | 原地哈希（值放到对应下标） | ⭐⭐⭐⭐ |
| LeetCode 169 多数元素 | Boyer-Moore 投票抵消 | ⭐⭐⭐⭐ |
| LeetCode 344 反转字符串 | 首尾双指针交换 | ⭐⭐⭐ |

识别信号：
- 题目要求"不使用额外空间" → 原地哈希 / 双指针
- 题目要求"找满足某条件的元组" → 排序 + 多指针
- "出现次数超过一半" → 投票算法

### 2. 链表

共性套路：虚拟头节点消除边界、快慢指针定位、三指针反转。

| 题目 | 核心技巧 | 面试频率 |
|------|----------|----------|
| LeetCode 206 反转链表 | 三指针迭代 / 递归 | ⭐⭐⭐⭐⭐ |
| LeetCode 141 环形链表 | Floyd 快慢指针判环 | ⭐⭐⭐⭐⭐ |
| LeetCode 21 合并有序链表 | dummy 头 + 双指针归并 | ⭐⭐⭐⭐⭐ |

识别信号：
- "反转" → 保存 next → 改指向 → 前移
- "是否有环" → fast 走 2 步、slow 走 1 步
- "合并/排序" → 归并思想 + 虚拟头

### 3. 栈的经典模型

共性套路：栈用于**维护"最近的未完成状态"**。单调栈用于 O(n) 求"左/右第一个更大/更小"。

| 题目 | 核心技巧 | 面试频率 |
|------|----------|----------|
| LeetCode 20 有效括号 | 左括号压右括号，弹出比较 | ⭐⭐⭐⭐⭐ |
| LeetCode 155 最小栈 | 辅助栈同步压入当前最小值 | ⭐⭐⭐⭐ |
| LeetCode 232 用栈实现队列 | 双栈倒腾（均摊 O(1)） | ⭐⭐⭐⭐ |
| LeetCode 84 柱状图最大矩形 | **单调递增栈** + 哨兵 | ⭐⭐⭐⭐⭐ |

识别信号：
- "括号匹配/嵌套" → 栈
- "O(1) 获取最值" → 辅助栈同步
- "柱状图/温度/股票买卖" → 单调栈

### 4. 队列与滑动窗口

共性套路：**单调双端队列**维护窗口内最值，队头过期就弹出，队尾破坏单调性就弹出。

| 题目 | 核心技巧 | 面试频率 |
|------|----------|----------|
| LeetCode 239 滑动窗口最大值 | 单调递减双端队列 | ⭐⭐⭐⭐⭐ |
| 循环队列实现 | 取模运算判满判空 | ⭐⭐⭐ |

### 5. 排序

共性套路：O(n²) 排序理解交换/插入思想，O(nlogn) 排序掌握分治递归。

| 算法 | 复杂度 | 稳定性 | 核心思想 |
|------|--------|--------|----------|
| 冒泡排序 | O(n²) | 稳定 | 相邻比较交换 |
| 插入排序 | O(n²) | 稳定 | 将当前元素插入已排序部分 |
| 选择排序 | O(n²) | 不稳定 | 每轮选最小放前面 |
| 归并排序 | O(nlogn) | 稳定 | 分治 + 合并有序子数组 |
| 快速排序 | O(nlogn) | 不稳定 | 分区 + 递归（pivot 选择是关键） |

面试关注点：
- 能否手写快排并说清分区过程
- 归并排序除了排序还能解决"逆序对"等衍生问题
- 稳定性的含义和应用场景

### 6. 二分查找

共性套路：**有序性 + 边界收缩**。变体的关键在于 `low/high` 更新策略和返回值选择。

| 变体 | 描述 | 边界细节 |
|------|------|----------|
| 基础二分 | 精确查找 | `low <= high` |
| 首个等于 | 找第一个等于 target 的 | `mid == target` 时继续左缩 |
| 末个等于 | 找最后一个等于 target 的 | `mid == target` 时继续右扩 |
| 首个大于等于 | lower_bound | `mid >= target` 时记录并左缩 |
| 末个小于等于 | upper_bound | `mid <= target` 时记录并右扩 |

| 题目 | 核心技巧 | 面试频率 |
|------|----------|----------|
| LeetCode 69 x 的平方根 | 标准二分 + long 防溢出 | ⭐⭐⭐ |
| LeetCode 33 搜索旋转排序数组 | 判断哪半边有序，分情况缩区间 | ⭐⭐⭐⭐⭐ |

### 7. 树

共性套路：递归定义 → 子问题结构。"当前节点需要从左右子树拿什么信息"是解题关键。

| 题目 | 核心技巧 | 面试频率 |
|------|----------|----------|
| LeetCode 226 翻转二叉树 | 递归交换左右子树 | ⭐⭐⭐⭐ |
| LeetCode 98 验证 BST | 递归传递 (min, max) 上下界 | ⭐⭐⭐⭐⭐ |
| LeetCode 104 最大深度 | max(左深度, 右深度) + 1 | ⭐⭐⭐ |

### 8. 哈希表

共性套路：空间换时间，O(1) 查找。"两数之和"是万题之母。

| 题目 | 核心技巧 | 面试频率 |
|------|----------|----------|
| LeetCode 1 两数之和 | 一边遍历一边查HashMap | ⭐⭐⭐⭐⭐ |
| LeetCode 146 LRU 缓存 | HashMap + 双向链表 = 全 O(1) | ⭐⭐⭐⭐⭐ |

### 9. 高级算法思想

| 算法思想 | 核心识别信号 | 代表题目 |
|----------|-------------|----------|
| 分治 | "合并两个有序结果" | 逆序对统计（归并排序改造） |

## 二、跨题型共性模板

### 模板 1：双指针
```
排序 → left=0, right=n-1 → 向中间收缩
适用于：两数之和、三数之和、盛最多水的容器、反转
```

### 模板 2：单调栈
```
for 每个元素:
    while 栈非空 && 当前元素破坏单调性:
        弹出栈顶 → 计算结果（左边界=新栈顶，右边界=当前元素）
    压栈
适用于：柱状图最大矩形、每日温度、下一个更大元素
```

### 模板 3：BFS 层序
```
queue.offer(起点)
while queue 非空:
    size = queue.size()  // 固定本层大小
    for i in size:
        node = queue.poll()
        处理 node → 将邻居入队
    层数++
适用于：最短路径、树的层序遍历、多源BFS
```

### 模板 4：0-1 背包
```
for 每个物品 i:
    for j 从容量倒序到 weight[i]:  // 倒序防重复
        dp[j] = max(dp[j], dp[j-weight[i]] + value[i])
适用于：子集和、分成两组差最小、恰好凑出目标值
```
